# Commandes en ligne

Une commande en ligne est mis à disposition afin d'exécuter les configurations.

## Commande

```bash
php yii.php fractalCmsImportExport:import-export/index -name={name} -version={version} -isTest=1 -params='key:value, key:value'
```

### paramètres

* **name** : nom de la configration
* **version** : version de la configuration ciblée
* **isTest** : Activer le mode test
* **params** : Paramètres métiers supplémentaires
* **pathFile** : chemin du fichier si import
